# maintainers

This document lists the maintainers of the orla project. Maintainers are responsible for reviewing pull requests, managing releases, and guiding the project's direction.

## current maintainers list

- @jadidbourbaki

## getting in touch

Please file a [GitHub issue](https://github.com/harvard-cns/orla/issues).
Please see [SECURITY.md](SECURITY.md) for reporting security vulnerabilities

## becoming a maintainer

If you're interested in becoming a maintainer, please reach out to the current maintainers through GitHub issues.

## responsibilities

Maintainers are expected to:
- Review and merge pull requests
- Help triage issues
- Guide the project's technical direction
- Ensure code quality
- Release new versions
- Foster a welcoming and inclusive community
